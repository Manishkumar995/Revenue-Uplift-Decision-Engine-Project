# Revenue-Uplift-Decision-Engine-Project (SQL + PYTHON + POWER BI)

# ROI-Driven Customer Targeting Engine using Churn & Price Elasticity

##  Overview

This project builds an end-to-end data-driven system to optimize customer targeting using behavioral analytics and pricing strategy.

It integrates:

* Customer segmentation (RFM).
* Churn probability modeling.
* Price elasticity analysis.
* ROI-based decision engine.

---

##  Problem Statement

Businesses often apply discounts without understanding:

* Which customers are likely to churn
* How pricing impacts demand
* Whether discounts generate real profit

This project solves that by identifying customers where discounts maximize ROI.

---

##  Tech Stack

* SQL (data aggregation and feature engineering)
* Python (Pandas, NumPy, Matplotlib)
* Power BI (dashboard and visualization)

---

##  Methodology

### 1. RFM Segmentation

* Recency, Frequency, Monetary analysis
* Customer value identification

### 2. Churn Modeling

* Behavior-based churn probability
* Enhanced with income segmentation

### 3. Price Elasticity

* Log-based elasticity calculation
* Demand sensitivity estimation

### 4. Decision Engine

* Simulated discount impact
* Calculated Expected Value and ROI
* Identified high-value target customers
